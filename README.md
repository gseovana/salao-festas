# Salão de festas Lutilê
O Salão de Festas Lutilê é um estabelecimento localizado em João Monlevade e sede seu espaço para a realização de eventos em geral como casamentos, festas infantis, festas de 15 anos, formaturas, entre outros.

## Escopo do sistema
### Objetivo
O objetivo geral do software proposto é auxiliar os proprietários do salão no processo de agendamento de visitas, locação e pagamentos. Atualmente esses processossão feitos inteiramente de forma manual e trazem desafios na gestão do estabelecimento.
